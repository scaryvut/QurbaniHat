# 🐄 QurbaniHat

A modern Qurbani livestock marketplace web application built with Next.js. Users can browse, view, and explore animals for Qurbani with a clean and responsive UI.

---

## 🌐 Live Demo

👉 https://your-live-url.com

---

## 🎯 Purpose

QurbaniHat is designed to simplify the process of discovering and viewing livestock for Qurbani. It provides a fast, user-friendly interface where users can explore animals, check details like price, weight, breed, and location, and make informed decisions.

---

## ✨ Key Features

- 🐄 Browse Qurbani animals with detailed listings  
- 💰 View pricing, weight, breed, and location info  
- 🔝 Top-priced animals section (dynamic sorting)  
- 🔐 Authentication system (Sign Up / Sign In)  
- 👤 User profile with session handling  
- 🌈 Responsive UI for mobile, tablet, and desktop  
- 🎠 Animated marquee tips section  
- ⚡ Fast client-side rendering with Next.js App Router  

---

## 🧰 Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- Better Auth (`authClient`)
- React Toastify
- Next.js Image Optimization
